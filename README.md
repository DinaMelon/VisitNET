# VisitNET
# Экскурсия по НЭТ (Новый Экспериментальный Театр)

## Описание проекта

Данный проект представляет собой интерактивную экскурсию по Новый Экспериментальный Театр, реализованную в виде 3D-приложения на Unity.
Пользователь может перемещаться вокруг здания театра, изучать его архитектуру и получать информацию об объектах в формате цифровой экскурсии.
Проект разработан в рамках учебной курсовой работы.

## Основные возможности

* Перемещение вокруг театра
* Осмотр архитектуры и ключевых памятников
* Интерактивные элементы с информацией
* Управление от первого лица
* Поддержка Web-версии (для встраивания на сайт)

## Структура проекта

```
KursovMagistr/
├── Assets/                # Ассеты проекта (модели, сцены, скрипты)
├── ProjectSettings/      # Настройки Unity
├── Packages/             # Пакеты Unity
└── ...

KursovMagistr/Web/
└── Build/                # WebGL билд приложения
```
## Технологии

* Unity (Unity3D)
* C#
* WebGL (для веб-сборки)

## Запуск проекта

### В Unity:

1. Открыть папку `KursovMagistr` через Unity Hub

### В браузере:

1. Перейти в папку:

   ```
   KursovMagistr/Web/
   ```
2. Разместить содержимое на веб-сервере
3. Открыть `index.html` в браузере

## Управление

* **Q/E** — передвижение
* **H** — вернуться на главную
* **W** — скрыть информацию
* **ЛКМ** — взаимодействие с объектами

## Цель проекта

Создание интерактивной цифровой экскурсии для:

* популяризации театра
* демонстрации возможностей Unity
* изучения 3D-разработки и WebGL
